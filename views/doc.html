Abstract

This specification describes the BUS language, defined as a dictionary of named properties and classes using W3C's RDF technology.

BUS is a project devoted to information about properties of bus and it's operational routes using the Web. Regardless of whether information is in people's heads, in physical or digital documents, or in the form of factual data, it can be linked. BUS does not compete with other travel booking and information Web sites; rather it provides a simple approach in which different sites can tell basic parts about bus and it's routes.

Status of This Document

The BUS specification is produced as part of the BUS-USU project creation in November-2021. The properties inside will not be changed as this is only a simple project (and possibly one time use). The BUS RDF namespace URI is fixed and its identifier is not expected to change. But even if the RDF namespace URI is fixed, keep in mind that there is a possibility of us taking down this URI if it is no longer needed.
